# amp_webcli Project (Vite + React)

This project was created using Vite and React.

## Running the Project (Development Mode)

To run the project locally for development purposes, follow the steps below.

### 1. Cloning the Repository (First Time Only)

```bash
git clone <gitlab-repository-url>
cd amp_webcli
```

### 2. Installing Dependencies (First Time Only or After Update)

Make sure you have [Node.js](https://nodejs.org/) installed. Then, in the main project directory (`amp_webcli`) run the command:

```bash
npm install
```

This command will install all required packages defined in `package.json`.

### 3. Running the Development Server

To run the application in development mode with automatic reloading after code changes, use the command:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or another address indicated in the terminal).

## Available Scripts

The following scripts are available in this project:

-   `npm run dev`: Runs the Vite development server.
-   `npm run build`: Creates a production version of the application in the `dist` folder.
-   `npm run preview`: Runs a local server to preview the production version of the application after building it.
